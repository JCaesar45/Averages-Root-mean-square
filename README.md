# Super Nintendo RMS Calculator

## Overview

Super Nintendo RMS Calculator is a visually nostalgic, pixel-art styled web application inspired by the classic SNES interface. It computes the **Root Mean Square (RMS)** of user-provided numbers, demonstrating both accurate mathematical processing and a retro aesthetic that appeals to developers and gaming enthusiasts alike.

## Features

- User-friendly input for comma-separated numbers
- Instant calculation of RMS with a sleek, SNES-inspired interface
- Vibrant, pixel-style design with pixel fonts and neon colors
- Responsive layout optimized for desktop screens
- Modular, clean JavaScript code demonstrating best practices
- Easy to extend with more features like validation or animations

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling with a retro SNES theme
- **JavaScript (ES6)** for functionality with modular design
- **Google Fonts** for pixel-perfect fonts (`Press Start 2P`)

## How to Run Locally

1. Clone or download this repository:
   ```bash
   git clone https://github.com/yourusername/snes-rms-calculator.git
   ``
2. Navigate into the project directory:
   ```bash
   cd snes-rms-calculator
   ``
3. Open `index.html` in your preferred web browser:
   - Double-click the file or
   - Use a local server like `Live Server` extension in VSCode

## Usage Instructions

- Enter numbers separated by commas in the input box (e.g., `1, 2, 3, 4, 5`)
- Click the **"Calculate RMS"** button
- The result will be displayed below with a nostalgic SNES style
